# TripMate
TripMate is an  android based journey planner application,The purpose of developing this app is to help the user create a list of the number of places that the user wants to visit and when the user will start the route, optimized route will be displayed to the user. As the user starts navigating, the application will use Google Maps for real time Navigation .Also, with multiple locations in a list, the application will monitor the userâ€™s location and provide a notification when the user is in the proximity of any location.
